# Insight-GPT
InsightGPT is an interactive Streamlit web application that performs automatic
data cleaning, EDA, anomaly detection, and AI-powered business recommendations
using Groq’s ultra-fast LLMs (LLaMA-3.1, Mixtral, etc.).

Upload a CSV → get instant insights → receive structured recommendations in JSON or natural language.

Check the deployed website here: https://insight-gpt-dqwigt3abesmudqiuwgoup.streamlit.app/
